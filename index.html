You are about to learn CSS Selectors! Selectors are how you pick which element to apply styles to.

Exhibit 1 - A CSS Rule

                p {
                    color: red;
                    margin-bottom: 1.2rem;
                }

Here, the "p" is the selector. It means "apply the following styles to all p elements".

To play, type in a CSS selector in the editor below to select the correct items on the table. If you get in the right, you will advance to the next level.

Hover over the items on the table to see their names. You can also click on the items to select them.
